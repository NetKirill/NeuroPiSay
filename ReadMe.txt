Проект НейроПисарь v1.2 
представляет функционал по переводу mp4 файлов видео в wav формат, с возможностью вырезать необходимый участок и сохранить его отдельно. 
Далее этот аудиофайл можно преобразовать в txt файл для удобного анализа и хранения.
Crop+TOwav(interface).py – файл который позволяет перевести mp4 в wav формате и обрезать его по точкам начала и конца
NeuroMain(Interface).py –  это интерфейс который позволяет открыть wav файл и запустить его обработку, которая происходит очень небыстро (да-да ничего не зависло, это такая обработка, простите ? ??? ?). По итогу получается вывод информации в интерфейсе и сохраненный файл {name}_result.txt
TxtToLine.py – файл который нужен для исправления возможных ошибок сохранения файла (примерно в 20% случаев распознавания текста записывает в построчном формате, этот файл преобразует такой результат в единую строку)  
vosk-model-ru-0.42.7z – архив с используемой моделью (не успели залить на Git, но модель открытая и легко находится, можно скачать по ссылке {https://alphacephei.com/vosk/models})
BogDan_crop_(3_2_pers).wav – 50 сек. Тестовый участок, который может показать точность работы
BogDan_crop_(3_2_pers).txt – расшифрованная версия тестового примера
AllAboutWork.docx – краткое описание реализованных функций и т.п.
Еноты_Подмосковья_Main.pdf – презентация проекта
